# Code of Conduct

## Our Standard

Contributors are expected to keep interactions:
- respectful
- technical
- direct
- non-hostile

Critique code, behavior and assumptions. Do not attack people.

## Expected Behavior

- assume good faith unless there is clear evidence otherwise
- keep discussions focused on the repository and the change under review
- provide actionable feedback
- accept correction when evidence is clear
- avoid harassment, insults, threats or discriminatory behavior

## Unacceptable Behavior

- personal attacks
- sustained hostility
- sexualized language or unwanted advances
- doxxing or sharing private information
- deliberately disruptive behavior in issues, PRs or discussions

## Enforcement

Maintainers may remove, edit or reject contributions and discussions that violate this code of conduct.

## Scope

This code of conduct applies to:
- issues
- pull requests
- code review discussions
- repository-linked collaboration spaces
