# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for important technical choices.

Use ADRs when a decision:
- affects the long-term structure of the codebase
- changes a security or authorization boundary
- changes persistence or runtime contracts
- would otherwise require future contributors to reconstruct context from commit history
