# Release v0.2.0

## Summary

`v0.2.0` establishes the first contributor-ready open source baseline for DevSynapse AI.

This release consolidates:
- backend/frontend runtime alignment
- command execution safety improvements
- project-scoped mutation authorization
- conversation persistence and management
- LLM token and cost telemetry
- monitoring, dashboard and usage reporting
- contributor and governance documentation

## Highlights

### Assistant and command workflow

- improved command extraction and repair
- structured execution status and reason codes
- clearer blocked/failed/successful command UX
- degraded response behavior when the DeepSeek API is slow or unavailable

### Persistence and monitoring

- migration-managed SQLite schemas
- conversation rehydration with command state
- LLM usage persistence with token and cost telemetry
- cost reporting by conversation, period and project

### Admin and permissions

- per-user project mutation allowlist
- audit logging for administrative changes
- stronger explicit project attribution for newer conversation rows

### Documentation and open source readiness

- MIT license
- contributor guide
- security policy
- changelog
- code of conduct
- issue and PR templates
- product showcase and screenshot evidence for chat, dashboard, settings and admin flows
