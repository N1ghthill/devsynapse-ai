# Release v0.3.0

## Summary

`v0.3.0` removes local-machine hardcoding, adds streaming chat responses, introduces project-scoped working directories, and polishes the operator experience.

## Highlights

### Portable configuration

- `DEV_WORKSPACE_ROOT`, `DEV_REPOS_ROOT`, and `DEV_PROJECTS_JSON` environment variables replace hardcoded `/home/irving` paths.
- `ALLOWED_DIRECTORIES` and `KNOWN_PROJECTS` are now computed from settings at runtime.
- `default_execution_cwd` uses `$HOME` by default instead of a hardcoded user path.
- `.env.example` documents all new variables.

### Streaming chat (SSE)

- New `POST /chat/stream` endpoint streams DeepSeek responses via Server-Sent Events.
- Backend uses `httpx` with `stream=True` for token-by-token delivery.
- Frontend consumes the stream via `ReadableStream` for real-time text rendering.
- Command extraction and repair work identically to the non-streaming path.

### Project selector

- Chat page shows a dropdown populated from the `/projects` endpoint.
- Selected project is passed as `project_name` in chat and command-execution requests.
- Enables project-scoped context without manual typing.

### Working directory per project

- `bash` and `grep` commands now resolve the working directory from the selected project's path.
- Falls back to `default_execution_cwd` when no project is selected.

### Budget defaults

- Daily budget default: `$1.00` (was `$0.00` / disabled).
- Monthly budget default: `$20.00` (was `$0.00` / disabled).
- New users get budget alerts out of the box.

### Operator polish

- `Ctrl+Enter` / `Cmd+Enter` sends a message (in addition to `Enter`).
- `Shift+Enter` inserts a newline.
- Chat input auto-focuses on page load.

## Breaking changes

- None. Existing environment files without `DEV_WORKSPACE_ROOT` / `DEV_REPOS_ROOT` use `$HOME`-based auto-detection.

## Migration

Run `make install-dev` to pick up the new `httpx` dependency:

```bash
make install-dev
make migrate
make seed-users
```

## Verified

- Backend test suite: `make test`
- Backend lint: `make lint`
- Frontend build: `make frontend-build`
