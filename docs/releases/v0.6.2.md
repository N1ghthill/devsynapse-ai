# Release v0.6.2

## Summary

`v0.6.2` is a patch release for desktop visual polish and chat streaming
stability.

## Highlights

### Desktop Icon Polish

- The packaged Tauri tray now uses a dedicated white high-contrast icon instead
  of reusing the default window icon.
- The tray glyph is drawn larger with thicker strokes so it is readable beside
  other system tray applications.
- The browser favicon now uses a cropped DevSynapse brand image so the mark does
  not appear undersized in tabs.

### Chat Stability

- Streaming assistant text is buffered in short batches before React state
  updates, reducing visible chat jitter while responses arrive.
- The pending assistant message now owns its loading state, avoiding duplicate
  loading bubbles during active streams.
- Chat workbench styling was refined while preserving existing project scope,
  command review, reasoning, usage and conversation controls.

## Platform Support

The supported desktop downloads remain:

- Linux x86_64 `.deb`
- Linux x86_64 `.rpm`
- Windows x86_64 Tauri installer

The supported shell installer/update path remains Debian/Ubuntu-style Linux.
macOS packaging remains configured but unvalidated.

## Release Compliance

- Application version: `0.6.2`
- Release tag: `v0.6.2`
- Supported desktop targets: Linux x86_64 and Windows x86_64
- Supported shell installer target: Debian/Ubuntu-style Linux with `apt`
- Runtime state: config, SQLite data and logs remain outside the source checkout
- Database migration impact: none
- API contract impact: none

## Migration

Update normally:

```bash
devsynapse update
```

or pin this release:

```bash
devsynapse update --version v0.6.2
```

The tray icon update requires restarting the packaged Tauri application after
installing the updated build.
