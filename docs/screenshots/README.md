# Screenshots and Evidence

This directory is reserved for curated visual evidence related to the project, such as:
- UI screenshots used in documentation
- release screenshots
- workflow examples for contributors

Recommended naming:

```text
YYYY-MM-DD-chat-command-execution.png
YYYY-MM-DD-dashboard-llm-usage.png
YYYY-MM-DD-admin-project-permissions.png
```

Guidelines:
- prefer screenshots that demonstrate current behavior
- avoid exposing secrets, tokens or private local data
- keep file names descriptive and date-stamped
