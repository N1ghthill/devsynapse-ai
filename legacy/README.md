# Legacy Artifacts

This directory holds historical files kept only for reference while the project transitions from its original POC shape to the current modular architecture.

Current legacy items:
- `poc_devsynapse.py`: original standalone proof-of-concept script
- `dashboard.html`: deprecated static dashboard prototype
- `opencode_bridge_backup.py`: earlier bridge implementation retained for comparison

Rules:
- do not build new features on top of these files;
- do not reference them from the main runtime path;
- remove them once their historical value is no longer useful.
